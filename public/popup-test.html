Browser Settings Guide

Chrome

  1. Click the lock icon or site settings icon in the address bar
  2. Find "Popups and redirects"
  3. Select "Allow"
  4. Refresh this page

Firefox

  1. Click the shield icon in the address bar
  2. Turn off "Enhanced Tracking Protection" for this site
  3. Or go to Settings ‚Üí Privacy & Security ‚Üí Permissions ‚Üí Popups
  4. Add exception for localhost

Edge

  1. Click the lock icon in the address bar
  2. Go to "Permissions for this site"
  3. Find "Pop-ups and redirects"
  4. Select "Allow"

Alternative: Use Incognito/Private Mode
